# chrome-theme-gruvbox-zed

[English](#english) | [日本語](#日本語)

---

## English

A Google Chrome theme port of the Gruvbox color palette, specifically refined for the "Zed" text editor. It provides both dark and light variants.

### Directory Structure

- `gruvbox-dark/`: Theme files for Dark mode
- `gruvbox-light/`: Theme files for Light mode
- `tools/`: Development scripts and reference colors

### Installation

#### Load as an Unpacked Extension

1. Clone or download this repository as a ZIP file and extract it.
2. Open Google Chrome and navigate to `chrome://extensions/`.
3. Enable **Developer mode** by toggling the switch in the top-right corner.
4. Click the **Load unpacked** button in the top-left corner.
5. Select either the `gruvbox-dark` or `gruvbox-light` folder from this repository.

## 日本語

エディタ「Zed」向けに調整されたGruvboxのカラーパレットを、Google Chrome用に移植したテーマです。ダークモードとライトモードの2種類を提供します。

### ディレクトリ構成

- `gruvbox-dark/`: ダークモード用テーマ
- `gruvbox-light/`: ライトモード用テーマ
- `tools/`: 開発用スクリプトおよびカラー定義メモ

### インストール方法

### 開発者モードによる読み込み

1. 本リポジトリをクローンするか、ZIPファイルとしてダウンロードして展開します。
2. Google Chromeを開き、アドレスバーに `chrome://extensions/` を入力して拡張機能ページにアクセスします。
3. 画面右上にある「デベロッパー モード」のトグルスイッチをオンにします。
4. 画面左上に表示される「パッケージ化されていない拡張機能を読み込む」ボタンをクリックします。
5. 本リポジトリ内の `gruvbox-dark` または `gruvbox-light` のフォルダを選択して読み込みます。
